关于空间维度的思考

发表于 2016-01-20   |   分类于 随笔

首先，高一维空间都是由低一维空间连续运动而成。
高一维空间生活着低一维度生物。

零维空间

一个点

一维空间(x)

一条直线，两个方向

二维空间(x,y)

一个平面

三维空间(x,y,z)

没有时间的概念。假设现在的世界时间停止，就是一个三维世界！人类(动物)即是三维生物。

四维空间(x,y,z,time)

是三维世界在以时间为轴的轨道上组合而成。
即是我们理解的世界!

此处，时间可以理解为另一个一维空间。

In a recent talk I said something that upset a lot of people: that you could get smarter programmers to work on a Python project than you could to work on a Java project.

I didn’t mean by this that Java programmers are dumb. I meant that Python programmers are smart. It’s a lot of work to learn a new programming language. And people don’t learn Python because it will get them a job(trouble); they learn it because they genuinely（真正地） like to program and aren’t satisfied with the languages they already know.
