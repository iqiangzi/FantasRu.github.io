什么是java

Java是一种跨平台的面向对象的语言。

JDK:(Java Development Kit) Java 语言的软件开发工具包,包含编译器，jre等

JRE:(java runtime environment) 运行java时所需的环境，包含jvm

JVM:(java virtual machine) java虚拟机

java平台

java SE(J2SE)，标准版，是我们通常用的一个版本,用于桌面应用开发

java EE(J2EE)，企业版，web开发，服务器端程序

java ME(J2ME)，主要用于开发手持设备端的应用程序
