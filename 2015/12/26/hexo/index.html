关于hexo&&如何定制属于你的hexo

发表于 2015-12-26   |   分类于 学习

hexo

刚刚接触github.io前端主题页面实在是丑爆了。了解到hexo搭建博客主题。

Hexo出自台湾大学生 tommy351 之手(惭愧…)，是一个基于 Node.js 的静态博客程序，其编译上百篇文字只需要几秒。hexo生成的静态网页可以直接放到GitHub Pages,需要安装node js.

compare

同类octopress jekyll，长时间未更新
octopress读取速度慢

  • 不可思议的快速 ─ 只要一眨眼静态文件即生成完成
  • 支持 Markdown语法
  • 仅需一道指令即可部署到 GitHub Pages 和 Heroku
  • 已移植 Octopress 插件
  • 高扩展性、自订性
  • 兼容于 Windows, Mac & Linux
